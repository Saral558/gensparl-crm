# 🚀 Dinesh CRM
## 📌 Overview

Dinesh CRM ek simple aur efficient Point of Sale system hai jo business operations ko manage karne ke liye design kiya gaya hai — jaise sales, stock, delivery aur expense tracking.

---

## ⚡ Features

* 🔐 Role-based Login System (har staff ko apna dashboard)
* 💰 Sales Management
* 📦 Stock Tracking
* 🚚 Delivery Management
* 💸 Expense Tracking
* 📊 Clean aur simple UI

## 🛠️ Installation & Setup

### Step 1: Clone Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

### Step 2: Open Project

Project folder ko open karo (VS Code / Antigravity / kisi bhi editor me)

### Step 3: Run Project

Agar HTML project hai:

```bash
Open index.html in browser
```

Agar server based hai:

```bash
npm install
npm start
```
